# Still Goods

Simple goods for calm living.

---

## 🛍️ Overview

「Still Goods」は、ミニマルで落ち着いた雑貨をテーマにしたポートフォリオサイトです。  
シンプルで静かな暮らしに寄り添う商品をイメージして制作しました。

---

## 🔧 Features

- 商品一覧の表示（JSONデータを使用）
- カテゴリフィルター機能
- 価格順ソート機能
- モーダルによる商品詳細表示
- レスポンシブ対応（スマホ・PC）
- アクセシビリティ対応（キーボード操作・aria属性）

---

## 🖥️ Tech Stack

- HTML
- CSS（Flexbox / Grid）
- JavaScript（Vanilla JS）
- JSON（データ管理）
